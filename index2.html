Project 1

Enhanced Calendar App

Project designed by Diyor Abduvaliev, Benjamin Smull, and Eric Lee. This Enhanced Calendar App project we worked on promotes a daily routine of jotting ones notes down in a certain place in order to remind one of the tasks they have left to complete. It helps people who struggle with keeping everything down jotted on their head. Rather, they can quickly log in or register and instantly jot down the notes/note for the day or for next month or however long the person feels like. We got inspired to make this project throughout the school year realizing often that we in fact did struggle with keeping each homework assignment written down in our brains. This is why we decided to create this app in order to utilize and benefit ourselves and others facing through the same problem. The project itself took around 3-4 weeks, but then we also had to create a seperate slideshow(linked below) and even the portfolio website(linked on my github page). We believe that there are still vast improvements to be made, but are willing to share our code made in java and development in order to inspire others to create something throughout the summer. In addition, for the Github Repository for GUI in HTML we got our code that we made in java and then with the assistance of AI we converted the original java code to HTML and generate a webbrowser that others can view(for accessbility) without having to copy our code in Intellij to actually run it. We did this incase others did not have intellij to actually display the GUI through java. The HTML gui was just created in order for others to see an example of how it looks through github, however our code that we also linked can be implemented through Intellij to give the same results.
